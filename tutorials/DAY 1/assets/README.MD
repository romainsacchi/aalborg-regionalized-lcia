# Day 1 Assets

This folder stores small Day 1 exercise assets.
The main BAFU Excel inventory used in `D1-04` is already bundled in the repository at:

- `../../data/lci-bafu.xlsx`

Do not download or copy the BAFU workbook into this folder for the course.

Additional Day 1 foreground assets:
- `lci-carbon-fiber.xlsx`: foreground workbook used in `D1-06` and `D1-07`.
- `lci-carbon-fiber-with-db-parameters.xlsx`: optional modified foreground workbook used in `D1-10`.
- `lci_hydrogen_electrolysis.xlsx`: foreground workbook kept for Brightway import examples.
