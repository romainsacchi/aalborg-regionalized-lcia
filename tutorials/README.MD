# Tutorials index

Core course structure:

- `00_SETUP/`: run before Day 1 to verify installation, imports, and one editable Brightway calculation
- `DAY 1/`: creates the main Brightway project state used later in the course
- `DAY 2/`: assumes the Day 1 project and imported BAFU data are available
- `DAY 3/`: assumes the Day 1 and Day 2 project state is still available

Use [SCHEDULE.MD](../SCHEDULE.MD) for the timed agenda and notebook order.
Use [TROUBLESHOOTING.md](../TROUBLESHOOTING.md) when a setup command, kernel, file path, or calculation fails.
